---
description: Подтянуть последние изменения с origin/master
---

Выполни:

## 1. Fetch
- `git fetch origin`

## 2. Проверь текущую ветку
- `git branch --show-current`
- Если не на `master` — спроси, переключаться ли (или сделай `git checkout master`, если рабочее дерево чистое)

## 3. Pull
- `git pull --ff-only origin master`
- Если fast-forward не получается (есть локальные коммиты или расхождение) — НЕ делай `git reset --hard` и НЕ делай merge без подтверждения. Сообщи пользователю и спроси что делать.

## 4. Покажи свежие коммиты
- `git log --oneline -5`

## Финальный отчёт
Одна строка: до какого коммита подтянулся master, сколько новых коммитов.
